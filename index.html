ranDOM wisDOM

Random Wisdom is a Flask app that interacts with a Dynamo DB on the back end. Four ‘product boxes’ are generated at random from the database and displayed on the page. (Currently the boxes contain quotes - hence Random Wisdom - but any kind of content can be used; product images for example.) Users can add content of their own to the database, which is then included in the list of possible boxes to be displayed. All of these parameters are editable. Now a link.
